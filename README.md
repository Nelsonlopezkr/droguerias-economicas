# Droguerías Económicas

Plataforma e-commerce farmacéutica desarrollada como proyecto de Ingeniería de Sistemas.

## Características

- Catálogo dinámico de productos
- Múltiples laboratorios por producto
- Cambio automático de imágenes por laboratorio
- Sistema de carrito de compras
- Filtros avanzados de búsqueda
- Diseño responsive
- Integración con WhatsApp
- Publicación mediante GitHub Pages

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages
